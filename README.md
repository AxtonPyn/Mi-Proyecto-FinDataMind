FindataMind – Plataforma de Gestión y Análisis Financiero

FindataMind es un proyecto académico–personal orientado a la gestión y análisis de información financiera mediante una aplicación web.

La plataforma permite registrar ingresos y gastos, clasificar transacciones por categorías y visualizar la información a través de dashboards, facilitando el control financiero y la toma de decisiones.

El sistema utiliza una API REST para la comunicación entre el frontend y el backend, con una arquitectura separada que garantiza orden, escalabilidad y mantenimiento del código.

Tecnologías utilizadas:

-Node.js (Backend / API REST)

-JavaScript (Frontend)

-SQL (Base de datos)

Objetivo del proyecto:
Automatizar y simplificar la gestión financiera mediante el uso de una API y visualización clara de los datos.
